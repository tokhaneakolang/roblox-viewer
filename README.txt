ROBLOX PROFILE VIEWER - Mobile Friendly
=========================================

HOW TO USE:
1. Upload ALL files (index.html, style.css, script.js) to any free hosting service
2. Open index.html in any browser
3. Enter any Roblox username
4. View their profile picture, badges, groups, and info

NO LOGIN REQUIRED! Works completely free!

FREE HOSTING OPTIONS:
- GitHub Pages (https://pages.github.com/)
- Netlify (https://www.netlify.com/)
- Vercel (https://vercel.com/)
- 000WebHost (https://www.000webhost.com/)
- InfinityFree (https://infinityfree.net/)

TO HOST ON GITHUB PAGES (EASIEST & FREE):
1. Create GitHub account
2. Create new repository
3. Upload these 3 files
4. Go to Settings → Pages
5. Enable GitHub Pages
6. Your site will be live at: https://YOURUSERNAME.github.io/REPONAME

TO USE ON MOBILE:
1. Upload files to any free host
2. Open the URL on your phone
3. Works perfectly on iOS and Android!
